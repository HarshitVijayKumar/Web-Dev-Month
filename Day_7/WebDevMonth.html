This is a month dedicated to Web Dev. Every day I have to learn something new and add to this site
It began on 9 September.

What will follows are paragraphs about the things I learnt on a day and my thoughts on that day. What will be attached with everyday is a screenshot of how the site looked each day.

Day-1

Day-1 covered the basics of HTML, I learnt the layout of a webpage, learnt how to add paragraphs, images, links and how to control were those links open their webpages at.
I also learnt how to add borders although they were not working very well yet. I discovered titles. Anything you write there pops up when you hover over it, I have planned to use it to keep track of the elements and the day they were added

Day-2

Day-2 began with CSS and using a CSS file to change the whole style of the webpage. I settled on External CSS.
Then I learned how to make images into links and transformed my insta and blog link to an image.
I also attempted to make a button into a link but that part seemed to be trickier than it looked.
Than I created a bookmark and decided on the actual content of the site. The site will document the chages done
everyday and will have a photo of those changes attached alongside them.

Day-3

On Day-3, I learned image maps, I edited the cat picture to include a Sharengan. Click the eye for a surprise!
I also made the icon for the webpage which is a cat peeking at you!

Day-4

Day-4 was hard, it was hard to focus and complete the day. I was only able to add a table which had problems in adding the image in it and setting its size. The Web-Dev Month has also started consuming a lot of time and I am wondering to restrict the time I invest each day.
I was also able to design a navigation bar

Day-5

I finally fixed the table, the images finally fit into the columns better. I discovered Iframes and implemented them now using that to display the table fully.
I also added a list at the end of the webpage.
I also used JavaScript to add a credits button in the end.

Day-6

Learned about the meta tags and added them.
Modified the Layout,
